# vue-typescript-boilerplate
**Vue와 TypeScript을 base로 한 Boilerplate code**

## Tech Stack
Node v14 base

- Vue v2.6
- TypeScript v4
- Webpack v5
- Express v4
- Vue Router v3
- Vuex v3

## Usage
**Install dependencies**
```
npm install
```

**Build (Webpack Bundling)**
```
npm run build
```

**Run development mode**
```
npm run start (npm run dev)
```

**Run production mode**
```
npm run prod
```
